# Safe Lane 🚗

A Flask web application designed to help teens learn safe driving rules, practice test questions, and prepare for their driver’s license.

## Features
- Practice driving test questions
- Learn road signs and traffic rules
- User-friendly interface for teens
- Built with Flask and Python

Create a folder to store the components e.g: safe-lane

## Installation
pip install -r requirements.txt
Clone the repository:


```bash
git clone https://github.com/jonuoha60/safe-lane.git
cd safe-lane

## Run application
